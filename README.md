# Adventure-Works-Report
Adventure Works Sales Report (Power BI)
Overview

Interactive Power BI dashboard built using the AdventureWorks dataset to analyze sales performance, customer behavior, and product trends.

Key Features

Tracks core KPIs: revenue, profit, orders, and return rate

Sales analysis by product, customer, and region

Trend analysis over time with interactive filtering

Insights into top products, regions, and customer segments

Tools & Data

Power BI

AdventureWorks dataset (sales, products, customers, calendar)
